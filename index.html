Deep learning architectures for system identification and control

Created a hybrid dynamic model for the UAM using deep learning architectures (FNN+TCN) for precise system identification and control

Completed

Accomplishments
  • Tools: Python, C++, ROS, Pytorch, Gazebo, Numpy
  • Unmanned aerial manipulators (UAMs) pose challenges for real-time optimal control due to the dynamic coupling between the UAV and manipulator. This work introduces a hybrid framework for UAM modeling and trajectory tracking control using platform data. A feedforward neural network (FNN) identifies the UAV's nominal model and coupling dynamics, while a deep quantile regression model, enhanced by a temporal convolutional network (TCN), captures uncertainties with explicit probability distributions. Leveraging these models, the framework combines linear MPC for translational dynamics and tube-based MPC for rotational dynamics, ensuring stability and robustness. The approach is validated through simulations and experiments.

Adaptive robust switching controller for tilt-rotor UAVs

Providing a step-by-step design of an optimized robust switching controller to improve the robustness and gaurantee precise path planning for tilt-rotor UAVs.

Completed

Accomplishments
  • Tools: Python, C++, Matlab, Simulink
  • Tilt-rotor UAVs have highly nonlinear dynamics due to contributions from both the fuselage and tilting rotors, complicating control design. Traditional logic-based controllers for flight mode transitions or input mapping often lack energy efficiency and struggle with fast trajectory tracking under disturbances. We presented a dynamic formulation and a two-layer constrained MPC strategy for trajectory tracking. The translational dynamics are managed by a constrained LMPC, formulated as a quadratically-constrained quadratic problem, ensuring feasible set-point solutions for rotational control. An optimizer generates a thrust vector, enabling seamless transitions between fuselage and rotor rotations. The rotational dynamics are approximated with piecewise affine subsystems, and a robust switching MPC with mode-dependent dwell time ensures stability, robustness, and actuator constraint integration.
